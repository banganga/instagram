Install
=====================

Make sure `config.yml` file is setup properly. [Docs here](http://shopify.github.io/themekit/configuration/).

#### 1. Install tools (gulp)
It will compile scss files from `/scss` to `/assets` directory
```
npm install
```

#### 2. Run gulp watch
```
gulp watch
```

#### 3. Run Shopify theme kit
```
theme watch
```
